# Contributing to Subterfuge

Thank you for your interest in helping improve Subterfuge! This repository is specifically for **bug reports only**.

## How to Contribute

### Bug Reports

This is the right place for bug reports. Use the [Bug Report template](https://github.com/gameshovel/subterfuge-issues/issues/new?template=bug_report.yml) to submit issues.

**Good bug reports include:**
- Clear, descriptive title
- Detailed steps to reproduce the issue
- Your device and game version information
- Screenshots or videos showing the bug
- Expected vs. actual behavior

### Feature Requests & Balance Feedback

**Do NOT submit feature requests or balance feedback here.**

Please share these on our [Discord server](https://discord.gg/TGZ2kFTDQV) instead, where you can:
- Discuss ideas with the community
- Get feedback on your suggestions
- Vote on other players' ideas
- Engage directly with the development team

### General Discussion

Join our [Discord](https://discord.gg/TGZ2kFTDQV) for:
- Strategy discussions
- Finding games and players
- Community events
- General questions and chat

## What This Repository is NOT For

Please do not use GitHub issues for:

| Topic | Where to Go |
|-------|-------------|
| Feature requests | [Discord](https://discord.gg/TGZ2kFTDQV) |
| Balance feedback | [Discord](https://discord.gg/TGZ2kFTDQV) |
| General questions | [Discord](https://discord.gg/TGZ2kFTDQV) |
| Account issues | support@secretplan.com |
| Purchase/billing problems | support@secretplan.com |
| Security vulnerabilities | See [SECURITY.md](SECURITY.md) |

## Issue Guidelines

### Do

- Search existing issues before creating a new one
- Use the provided bug report template
- Provide as much detail as possible
- Include screenshots or video when possible
- Be respectful and patient

### Don't

- Submit duplicate issues
- Submit feature requests or balance suggestions
- Include personal attacks or inappropriate language
- Share personal information of other players
- Submit issues unrelated to bugs

## Response Time

Our team reviews bug reports regularly. While we read every issue, we may not respond to each one individually. High-priority bugs affecting gameplay will be addressed first.

## Questions?

If you're unsure whether something is a bug or have questions about the game, please ask on [Discord](https://discord.gg/TGZ2kFTDQV) first.

Thank you for helping make Subterfuge better!
